# duoPing
为了批量测速Vultr机房线路质量写的脚本

用过几次海外机房才知道，并不是延迟越短的线路越好。
vultr的日本机房延迟很低，但用起来总感觉不太爽，一测才知道 好家伙丢包率50%+

这个脚本专门用来测试线路延迟、稳定性的，17个机房同时测速，想ping多少次就直接输入多少次，个人自用方便一点
